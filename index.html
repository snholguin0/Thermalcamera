Live demo

  1. Place hand 3–5 inches above the sensor.
  2. Move your hand slowly so the heat has time to show up.
  3. Use the buttons to switch between demo modes.
  • Mode 0 Simple thermal view
  • Mode 1 Heat pen drawing
  • Mode 2 Laser grid + zones

How It Works

Hardware

  • Raspberry Pi 4 Model B
  • Adafruit AMG8833 8×8 thermal camera
  • Solderless breadboard + jumper wires

Software

  • Python + I²C to read the AMG8833
  • NumPy + SciPy for processing and smoothing
  • Flask server to stream frames to the website

Modes

Mode 0 – Thermal View
A clean heat map of the 8×8 sensor, upscaled for smoother color gradients.

Mode 1 – Heat Pen
The hottest part of your hand becomes a “pen.” Moving your hand draws bright blobs that slowly fade.

Mode 2 – Laser Grid
The image is split into four zones. When a zone warms past the background, its square turns red and the text shows which zones were “tripped.”
