Live Thermal View

This panel shows the live output from the thermal camera running on the Raspberry Pi. Use the mode buttons to switch between views on the Pi.

Mode 0: Simple thermal view

The Pi listens for keyboard input directly, so these buttons only change the label on the website. On the Pi, press m to cycle modes.

live demo

  1. place hand 3–5 inches above the sensor.
  2. Move your hand slowly so the heat has time to show up.
  3. Switch modes on the Pi with the keyboard:
  • m – change mode (0 → 1 → 2)

The web page just mirrors whatever the Pi is currently displaying.
