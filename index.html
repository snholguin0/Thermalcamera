How It Works

Hardware

  • Raspberry Pi 4 Model B
  • Adafruit AMG8833 8×8 thermal camera
  • Solderless breadboard + jumper wires

Software

  • Python to read the AMG8833 over I²C
  • NumPy + SciPy for processing and smoothing
  • matplotlib for the color heat map
  • Three demo modes: thermal, heat pen, laser grid

Modes

Mode 0 – Thermal View

A clean heat map of the 8×8 sensor, upscaled for smoother color gradients. Great for showing the basic idea of thermal imaging and how your hand heats up the scene.

Try Mode 0 Live

Mode 1 – Heat Pen

The hottest part of your hand becomes a “pen.” Move your hand to draw bright blobs that slowly fade away. Kids can “paint” on the screen with heat.

Try Mode 1 Live

Mode 2 – Laser Grid

The image is split into four zones. When a zone warms up past the background, its square turns red and the text shows which zones were “tripped.”

Try Mode 2 Live
